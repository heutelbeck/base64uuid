# Convenience class for compactly represented UUIDs encoding the 32bytes as Base64

Usage example: `Base64Id.randomID()`

## Dependencies:

```
<dependency>
  <groupId>com.heutelbeck</groupId>
  <artifactId>base64uuid</artifactId>
  <version>1.0.0</version>
</dependency>
```
